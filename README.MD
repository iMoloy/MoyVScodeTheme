# Moy Theme

An elegant and flavored dark theme for VS Code, customized with subtle borders and optimized for maximum eye comfort during long coding sessions.

![theme screenshot one](images/preview.png)

## Features (v2.0.0)

- **Eye-Comfort Dark Background**: The signature dark blue-slate background (#070E13) provides a deep, calming atmosphere that minimizes eye strain and screen glare.
- **Crisp Text legibility**: Uses a soft slate-white color (#f1f5f9) instead of pure white to prevent halation and eye fatigue.
- **Highly Legible Comments**: Brightened comment text (#828a91) ensures you can read document notes comfortably.
- **Solid Highlight Accents**: Key code syntaxes are colored with solid, rich, and high-quality colors:
  - **Keywords**: Solid Crimson Red (`#ff3b30`)
  - **Functions/Calls**: Solid Lime Green (`#4ce625`)
  - **Constants/Numbers**: Solid Purple (`#be3df4`)
  - **Types/Classes/Tags**: Solid Sky Cyan (`#00d2fc`)
  - **Parameters**: Solid Warm Orange (`#ff6b22`)
  - **Active Tab/Status**: Solid Golden Yellow (`#ffcc00`)
- **Clear File Navigation**: Sidebar and navbar file names are styled with a clean light grey (#cbd5e1) for stress-free folder browsing.

## Installation

### From VS Code Extension Marketplace
1. Open the **Extensions** sidebar in VS Code (`Ctrl+Shift+X` or `Cmd+Shift+X`).
2. Search for `Moy Theme`.
3. Click **Install**.
4. Navigate to `File > Preferences > Color Theme` and select **Moy Theme**.

### Manual Installation (VSIX)
1. Download the latest **`moy-theme-2.0.0.vsix`** file.
2. Open the **Extensions** sidebar in VS Code.
3. Click the `...` (More Actions) icon in the top-right corner of the Extensions panel.
4. Select **Install from VSIX...**.
5. Choose the downloaded `moy-theme-2.0.0.vsix` file.
